install llvm
    Make sure to select "Add LLVM to the system PATH for all users".
    On my computer, codeblocks crashes without this, because llvm comes with
    a DLL file that codeblocks needs (api-ms-win-crt-string-l1-1-0.dll)

install codeblocks standard install is good? todo make sure.

add to path: C:\Program Files\CodeBlocks\MinGW\bin

start codeblocks
	- set clang compiler as default (scroll down)

Set dark theme
    download theme.conf from https://github.com/virtualmanu/Codeblocks-Themes/blob/master/theme.conf
    close codeblocks
    open "Codeblocks Share Config" from start menu
    for destination, select "default.conf" in the directory where dialog opens by default
    for source, select downloaded theme.conf file
    check all boxes at left
    click "Transfer"
    click "Save"
    click "Close"
    open codeblocks
    menubar --> Settings --> Editor --> Syntax highlighting --> click "Add" button
    Browse to downloaded theme.conf

if you installed codeblocks or LLVM to non-standard location, edit jou.cbp as appropriate
open project
click the build button
